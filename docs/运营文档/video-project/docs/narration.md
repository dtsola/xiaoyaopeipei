## 免费TTS服务推荐

### 1. Edge TTS（推荐）

- **优势**：通义千问同生态，音质佳
- **免费额度**：新用户有免费额度

### 3. 讯飞TTS（免费版）
- **在线地址**：https://www.xfyun.cn/services/voicedictation

## 使用 Edge TTS 生成配音

### 方法1：在线工具（推荐）

1. 访问 https://tts-vue.com/
2. 选择语音：`zh-CN-XiaoxiaoNeural`（女声）或 `zh-CN-YunyangNeural`（男声）
3. 粘贴文案，生成下载

### 方法2：Python 脚本

```bash
pip install edge-tts
```

```python
import edge_tts

TEXT = """
24小时接单，可能吗？
很多电脑店老板都有一个头疼的问题...
"""

async def generate_narration():
    communicate = edge_tts.Communicate(
        text=TEXT,
        voice="zh-CN-XiaoxiaoNeural",  # 女声
        # voice="zh-CN-YunyangNeural",  # 男声
        rate="+0%",     # 语速
        volume="+0%"    # 音量
    )
    await communicate.save("narration.mp3")

if __name__ == "__main__":
    import asyncio
    asyncio.run(generate_narration())
```

## 音频文件组织

生成后放入 `public/audio/` 目录：

```
public/audio/
├── narration.mp3      # 完整解说（90秒）
├── bgm.mp3           # 背景音乐（可选）
└── sfx/              # 音效（可选）
```

## 参数建议

- **语音**: `zh-CN-XiaoxiaoNeural` (女) 或 `zh-CN-YunyangNeural` (男)
- **语速**: `+0%` (正常)
- **音量**: `+0%`
- **格式**: MP3
- **采样率**: 44100Hz
