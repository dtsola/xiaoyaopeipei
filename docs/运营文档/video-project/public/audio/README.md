# 音频文件说明

## 需要添加的音频

1. **narration.mp3** - 解说配音（必需）
   - 使用 Edge TTS 生成
   - 时长：90秒
   - 参考：[配音文案](../../docs/narration.md)

2. **bgm.mp3** - 背景音乐（可选）
   - 选择轻快的背景音乐
   - 注意版权问题

## 快速生成配音

### 在线工具（最简单）

1. 访问 https://tts-vue.com/
2. 选择语音：`zh-CN-XiaoxiaoNeural` 或 `zh-CN-YunyangNeural`
3. 粘贴以下文案：

```
24小时接单，可能吗？
很多电脑店老板都有一个头疼的问题。客户随时来问，不可能一直守着手机，回复慢一点，客户就跑了。
于是我用 FastAPI、Vue 3 和通义千问，做了一个 AI 对话式导购助手。核心思路很简单——用 AI 帮小商家 24 小时自动接单。
客户扫码进入对话页，自然语言描述需求。AI 自动识别意图，推荐高配、中配、性价比三套方案。还可以多方案对比，配置差异一目了然。客户留下联系方式，商家收到通知及时跟进。
商家后台能看到完整的访问量、线索量趋势，以及所有客户线索和对话历史。配置 SKU 也是增删改查，管理很方便。生成专属推广二维码，随时分享获客。
项目已经开发完成，我决定完全开源，永久免费。Docker 一键启动，几分钟就能跑起来。GitHub 链接在简介，欢迎试用和反馈。
这个项目是技术实践，也希望能帮到有需要的同行。欢迎交流技术话题或合作机会。
```

4. 生成后下载，重命名为 `narration.mp3`，放入此目录

## Python 脚本生成

```bash
pip install edge-tts
python generate_narration.py
```
