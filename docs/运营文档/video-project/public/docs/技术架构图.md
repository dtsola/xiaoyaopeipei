## 数据流图

```mermaid
sequenceDiagram
    participant User as 👤 用户
    participant Vue as 📱 Vue前端
    participant API as ⚡ FastAPI
    participant AI as 🤖 通义千问
    participant DB as 🗄️ MySQL

    User->>Vue: 自然语言描述需求
    Vue->>API: POST /api/user/chat/message
    API->>AI: 调用LLM意图识别
    AI-->>API: 返回提取的信息
    API->>DB: 查询匹配的SKU
    DB-->>API: 返回配置列表
    API-->>Vue: 返回推荐方案
    Vue-->>User: 展示高/中/性价比方案

    User->>Vue: 选择方案+联系方式
    Vue->>API: POST /api/user/lead/submit
    API->>DB: 保存线索信息
    API-->>Vue: 提交成功
    Vue-->>User: 展示商家信息
```
